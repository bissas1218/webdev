ATMEL ATmega328P Data Sheet

8-bit AVR Microcontroller with 4/8/16/32K Bytes In-System Programmable Flash

소개

Atmega328은 저전력이다. AVR은 향상된 RISC구조 기반의 CMOS 8비트 마이크로컨트롤러이다.

단일 클럭 주기로 명령을 실행함으로써 장치는 메가헤르츠당 초당 100만 개에 가까운 CPU 처리량을 달성하여 시스템 설계자가 처리 속도 대비 전력 소비를 최적화할 수 있다.

특징들

● 고성능, 저전력 AVR 8-비트 마이크로컨트롤러 제품군

● 향상된 축소명령집합컴퓨터,고정길이명령어(RISC) 구조이다

 - 131개의 강력한 명령어들 - 대부분 단일클럭 사이클로 실행

 - 32 * 8 범용 작업 레지스터

 - 완전 정적 작동

 - 20MHz에서 최대 20MIPS(100만 명령/초) 처리량

 - 온칩 2사이클 승수

● 고내구성 비휘발성 메모리 세그먼트(단편,조각,구분,부분)

 - 4/8/16/32K 바이트 시스템 자체 프로그래밍 가능 플레시 프로그램 메모리

 - 256/512/1K 바이트 EEPROM(비휘발성 메모리)

 - 512/1K/2K 바이트 내부 SRAM(정적메모리)

 - 쓰기/지우기 주기: 10,000 플레시 / 100,000 EEPROM

 - 데이터 보존: 85℃에서 20년 / 25℃에서 100년

 - 독립잠금 비트가 있는 선택가능한 부팅코드 섹션

   온칩 부트 프로그램에 의한 인시스템 프로그래밍

   진정한 읽기 및 쓰기 작업

 - 소프트웨어 보안을 위한 프로그래밍 잠금
